Template Items

Template item descriptions which will be rendered as HTML components at runtime.

Resource definition

{{ '{' }}
"id": integer,
"name": string,
"type": string,
"sub_type": string,
"position": integer,
"required": boolean,
"aux_attributes":
{
"question": { "value": string, "data_source": string },
"max_length": integer,
"min_length": integer,
"placeholder": string,
"src": string
}
"created_at": string,
"updated_at": string
{{ '}' }}
 

Template Items: list

Lists the items by template ID.

Request

GET /v1/template-items/?template_id={TEMPLATE_ID}

Response

{{ '{' }}
"count": integer,
"next": string,
"previous": string,
"results": [
Template Item Resource
]
{{ '}' }}
Field Name Data Type Description
count integer The number of templates that this user owns.
next string URL to fetch next page.
previous string URL to fetch previous page.
results [ ] list The list template items.
 

Template Items: retrieve

Retrieve a template item by ID.

Request

GET /v1/template-items/templateItemId/

Response

This method will return a Template Item Resource.
 

Template Items: create

Create a new template item.

Request

POST /v1/template-items/

Request Data

{{ '{' }}
"name": string,
"type": string,
"sub_type": string,
"predecessor": integer,
"required": boolean,
"template": integer,
"aux_attributes":
{
"question": { "value": string, "data_source": string },
"max_length": integer,
"min_length": integer,
"placeholder": string,
"src": string
}
{{ '}' }}
Field Name Data Type Description
name string A name for this template item.
type string Item type which maps to an HTML element, accepted values are: text, instructions, file_upload, iframe, audio, image, radio, select_list, checkbox
predecessor integer The id of the item which comes before this item (the predecessor) which determines the position in the template.
sub_type string If type is set to "text", you can provide a sub type for "number", "text_area" and "text" for a number field, a large text area and a regular text field respectively. Else it can be null.
required boolean A flag which is used to make a field required or optional.
template integer Id of the template this item is being added to.
aux_attributes JSON object These properties vary depending on the type of the item, meaning that it is an unstructured object.
aux_attributes.question.value string A question or header to be added at the top of the item, it can be static text or a key of the task data that you provide (use {} syntax for that).
aux_attributes.max_length integer Optional: applies to items of type text, it defines the maximum number of characters that the user can enter as a response.
aux_attributes.min_length integer Optional: applies to items of type text, it defines the minimum number of characters that the user can enter as a response.
aux_attributes.placeholder string Optional: applies to items of type text, a placeholder for the text field.
aux_attributes.src string Required for iframe, audio, image: the url source for this element, it can be static or a variable coming from your task data, or a mix of both.
aux_attributes.layout string Required for items of type checkbox and radio: accepted values are "row" and "column" which sets the orientation of the options.
aux_attributes.shuffle_options boolean Required for items of type checkbox, radio and select_list: if set to true the options will be shuffled randomly every time they displayed to the user.
aux_attributes.options list of objects Required for items of type checkbox, radio and select_list: a list of options which will be used to generate the html element.
aux_attributes.options.value string Value of the selection option which will be displayed to the user, this will also be the value of the html element for that option, variables can also be used here.
aux_attributes.options.position integer Position of this option in the list.

Response

{{ '{' }}"id": integer{{ '}' }}
 

Template Items: destroy

Delete a template item by ID.

Request

DELETE /v1/template-items/templateItemId/

Response

The response for this request will be empty.
