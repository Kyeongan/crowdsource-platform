Tasks

Data entries which are used to populate the project template.

Resource definition

{{ '{' }}
"id": integer,
"group_id": integer,
"hash": string,
"data": json object,
"price": float,
"created_at": string,
"updated_at": string
{{ '}' }}
 

Tasks: list

Lists the projects by project.

Request

GET /v1/tasks/

Request Parameters

Field Name Data Type Description
project_id integer Project ID to fetch the tasks for.

Response

{{ '{' }}
"count": integer,
"next": string,
"previous": string,
"results": [
Task Resource
]
{{ '}' }}
Field Name Data Type Description
count integer The number of tasks belonging to this project.
next string URL to fetch next page.
previous string URL to fetch previous page.
results [ ] list The list of tasks.
 

Tasks: retrieve

Retrieve a task by ID.

Request

GET /v1/tasks/taskId/

Response

This method will return a Task Resource.
 

Tasks: create

Create a new task.

Request

POST /v1/tasks/?project_id={PROJECT_ID}

Request Data

{{ '{' }}
"data": json object,
"price": float
{{ '}' }}
Field Name Data Type Description
data json object The data which will be used in the template to render the task.
price float Optional: Price for this specific task, only relevant if custom pricing is used, otherwise Project.price will be used.
project_id as url parameter integer Id of the project which this task is being created for.

Response

{{ '{' }}"id": integer{{ '}' }}
 

Tasks: destroy

Delete a task by ID.

Request

DELETE /v1/tasks/taskId/

Response

The response for this request will be empty.
 

Tasks: results

Get all the assignment results for this task.

Request

GET /v1/tasks/taskID/assignment-results/

Response

{{ '{' }}
"count": integer,
"next": string,
"previous": string,
"results": [
Assignment Resource
]
{{ '}' }}
