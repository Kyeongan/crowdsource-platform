You should use the sandbox instance for testing https://sandbox.daemo.org
Always set these two HTTP headers: "Content-Type: application/json", and "Authorization: Bearer YOUR_ACCESS_TOKEN".

Creating a project

POST /v1/projects/
{{ '{' }}
"name": "Test Project",
"price": 0.1,
"repetition": 1,
"timeout": 120,
"template": {{ '{' }}
"name": "My Template",
"items": [] // empty for now
{{ '}' }},
{{ '}' }}

Response

{{ '{' }}
"id": 1,
"name": "Test Project",
"price": 0.1,
"timeout": 120,
"template_id": 1
{{ '}' }}
 

Adding items to the template

POST /v1/template-items/
{{ '{' }}
"name": "Item 1",
"type": "text",
"type": "text",
"required": true,
"template": 1,
"predecessor": null,
"aux_attributes":
{
"question": { "value": "Write a summary this article {{ '{' }}{{ '{url}' }}{{ '}' }}", "data_source": null },
"max_length": 128,
"min_length": 16,
"placeholder": "Enter your response here"
}
{{ '}' }}

Response

{{ '{' }}
"id": 1,
...
{{ '}' }}
 

Adding data (tasks) to a project

Request

POST /v1/tasks/?project_id=1

Request Data

{{ '{' }}
"data": {"url": "https://en.wikipedia.org/wiki/Enigma_machine"}
{{ '}' }}

Response

{{ '{' }}"id": 1{{ '}' }}
