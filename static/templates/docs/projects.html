Projects: list

Lists the projects by account type.

Request

GET /v1/projects/?account_type={worker|requester}

Response

{{ '{' }}
"count": integer,
"next": string,
"previous": string,
"results": [
Projects Resource
]
{{ '}' }}
Field Name Data Type Description
count integer The number of all existing projects
next string URL to fetch next page.
previous string URL to fetch previous page.
results [ ] list The list of projects.
 

Projects: retrieve

Retrieve a project by ID.

Request

GET /v1/projects/projectId/

Response

This method will return a Project Resource.
 

Projects: create

Create a new project.

Request

POST /v1/projects/

Request Data

{{ '{' }}
"name": string,
"price": float,
"timeout": integer,
"template": {{ '{' }}
"name": string,
"items": [
TemplateItem Resource,
...
]
{{ '}' }},
{{ '}' }}
Field Name Data Type Description
name string Project name.
price float Price for a single task in this project.
timeout integer The time in minutes that a worker is allowed to keep this task in an accepted state.
enable_boomerang boolean A flag which is used to enable or disable the cascaded release of the project.
template JSON object Optional: Template specification for this project, name and template items included.

Response

{{ '{' }}"id": integer, "template_id": integer{{ '}' }}
 

Projects: destroy

Delete a project by ID.

Request

DELETE /v1/projects/projectId/

Response

The response for this request will be empty.
 

Projects: publish

Publish a project.

Request

POST /v1/projects/projectId/publish/

Request Data

None

Response

{{ '{' }}"message": "Project :projectName successfully published"{{ '}' }}
