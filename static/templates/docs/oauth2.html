OAuth2 Authentication

Daemo supports authentication via Oauth2 for all REST API calls.

How to get the tokens

Download the tokens as JSON by navigating to your Daemo Profile and clicking "Get Credentials" under the more_vert menu.
 

How authenticate when making API calls

Set the HTTP header "Authorization" to "Bearer your_access_token".
 

OAuth2: refresh_token

Use an existing valid refresh_token to get a new access_token.

Request

POST /api/oauth2-ng/token/

Request Data

{{ '{' }}
"client_id": string,
"grant_type": "refresh_token",
"refresh_token": string,
{{ '}' }}
Field Name Data Type Description
client_id string Client ID from the file initially downloaded from Daemo.
grant_type string Always use "refresh_token".
refresh_token string A valid refresh_token.

Response

{{ '{' }}
"access_token": string,
"refresh_token": string
{{ '}' }}
